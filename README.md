# Portfolio Blog Starter

This is a porfolio site template complete with a blog. Includes:

- MDX and Markdown support
- Optimized for SEO (sitemap, robots, JSON-LD schema)
- RSS Feed
- Static OG image metadata
- Syntax highlighting
- Tailwind v4
- Geist font

## How to Use

Install dependencies, then run Next.js in development mode:

```bash
npm install
npm run dev
```
